# Шаблоны

Все HTML шаблоны находятся в директории `templates/tourism/` и используют систему наследования шаблонов Django.

## Базовый шаблон

### `base.html`

Базовый шаблон, от которого наследуются все остальные шаблоны.

#### Основные компоненты:

1. **Заголовок**
   - Bootstrap 5.1.3 CSS
   - Font Awesome 6.0.0 для иконок
   - Адаптивный viewport
   - Кастомные стили для hover-эффектов и layout

2. **Навигационное меню**
   - Брендинг с иконкой самолета
   - Ссылки на основные разделы
   - Отображение имени пользователя для авторизованных
   - Кнопки входа/регистрации для неавторизованных
   - Ссылка на статистику по странам
   - Административная ссылка для staff-пользователей

3. **Основной контент**
   - Блок `{% block content %}` для переопределения в дочерних шаблонах
   - Отображение системных сообщений Django messages

4. **Футер**
   - Копирайт и информация о проекте
   - Фиксированная высота для sticky footer

#### Использование:

```html
{% extends 'tourism/base.html' %}
{% block title %}Название страницы{% endblock %}
{% block content %}
    <!-- Содержимое страницы -->
{% endblock %}
```

## Основные шаблоны

### `tour_list.html`

Отображает список всех туров с поиском и пагинацией.

**Особенности:**
- Форма поиска по названию, стране и агентству
- Карточки туров с Bootstrap grid (3 колонки на десктопе)
- Hover-эффекты на карточках
- Пагинация через Bootstrap pagination
- Отображение цены и страны тура

### `tour_detail.html`

Детальная страница тура с отзывами.

**Особенности:**
- Полная информация о туре
- Кнопка резервирования (только для авторизованных)
- Список отзывов с рейтингом (звездочки)
- Пагинация отзывов
- Кнопка добавления отзыва (только для авторизованных)
- Форматирование дат

### `make_reservation.html`

Форма создания резервирования.

**Особенности:**
- Форма с полем количества участников
- Отображение информации о туре
- Валидация через Django forms
- Сообщения об ошибках

### `my_reservations.html`

Список резервирований текущего пользователя.

**Особенности:**
- Таблица с информацией о резервированиях
- Цветовая индикация статусов (badges)
- Кнопки редактирования и удаления
- Пагинация
- Фильтрация по статусу

### `edit_reservation.html`

Форма редактирования резервирования.

**Особенности:**
- Предзаполненная форма
- Валидация количества участников
- Кнопки сохранения и отмены

### `delete_reservation.html`

Подтверждение удаления резервирования.

**Особенности:**
- Подтверждающее сообщение
- Кнопки подтверждения и отмены

### `add_review.html`

Форма добавления отзыва о туре.

**Особенности:**
- Поле рейтинга (1-10)
- Текстовое поле комментария
- Поле выбора даты тура (HTML5 date picker)

### `login.html` и `register.html`

Формы аутентификации.

**Особенности:**
- Стандартные Django формы авторизации
- Стилизация Bootstrap
- Редиректы для уже авторизованных пользователей

## Административные шаблоны

### `admin_reservations.html`

Управление всеми резервированиями (только для staff).

**Особенности:**
- Фильтрация по статусу
- Кнопки подтверждения и отмены для каждого резервирования
- Таблица с полной информацией
- Пагинация (20 на страницу)

### `reservations_by_country.html`

Список резервирований по конкретной стране (только для staff).

**Особенности:**
- Фильтрация по стране
- Только подтвержденные резервирования
- Пагинация

### `tours_by_country.html`

Статистика по проданным турам по странам.

**Особенности:**
- Группировка туров по странам
- Подсчет количества проданных туров
- Таблица с сортировкой

## Использование Django Template Language

### Теги:

- `{% extends %}` - наследование шаблонов
- `{% block %}` - блоки для переопределения
- `{% url %}` - генерация URL через именованные маршруты
- `{% if %}` / `{% endif %}` - условные операторы
- `{% for %}` / `{% endfor %}` - циклы
- `{% csrf_token %}` - защита от CSRF атак

### Фильтры:

- `{{ value|date:"d.m.Y" }}` - форматирование дат
- `{{ value|floatformat:2 }}` - форматирование чисел
- `{{ user.is_authenticated }}` - проверка авторизации
- `{{ user.is_staff }}` - проверка staff статуса

### Примеры:

```html
{% if user.is_authenticated %}
    <a href="{% url 'tourism:my_reservations' %}">Мои резервирования</a>
{% else %}
    <a href="{% url 'tourism:login' %}">Войти</a>
{% endif %}
```

## Дизайн и стилизация

### Bootstrap 5 компоненты:

- **Cards** - для отображения туров
- **Tables** - для списков резервирований
- **Forms** - для всех форм ввода
- **Pagination** - для навигации по страницам
- **Badges** - для статусов и рейтингов
- **Navbar** - для навигации
- **Alerts** - для системных сообщений

### Адаптивность:

- Mobile-first подход
- Responsive grid система
- Collapsible навигация на мобильных устройствах

### Кастомные стили:

- Hover-эффекты на карточках туров
- Цветовая индикация статусов резервирований
- Sticky footer для корректного отображения на всех разрешениях
