# Задача 1 - UDP сервер и клиент

## Цель

Реализация базового взаимодействия между сервером и клиентом через протокол UDP (User Datagram Protocol).

## Выполненная работа

Был создан UDP сервер, который ожидает входящие сообщения на порту 8080 и отвечает на каждое полученное сообщение. Клиент отправляет сообщение "Hello, server" и получает ответ "Hello, client". 

В процессе реализации изучены особенности UDP протокола:
- Использование `socket.SOCK_DGRAM` для UDP соединений
- Методы `recvfrom()` и `sendto()` для работы с адресами отправителя и получателя
- Необходимость кодирования/декодирования UTF-8 для текстовых данных
- Обработка таймаутов для предотвращения бесконечного ожидания

## Результат

Успешно реализовано UDP взаимодействие между сервером и клиентом. Изучены преимущества UDP (низкая задержка, простота) и его недостатки (отсутствие гарантий доставки, возможная потеря пакетов).
