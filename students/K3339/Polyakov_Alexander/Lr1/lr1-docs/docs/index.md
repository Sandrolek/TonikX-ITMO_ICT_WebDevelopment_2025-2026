# Отчет по лабораторной работе 1 - Сетевые технологии

## Цель работы

Изучение различных сетевых протоколов и технологий в Python через реализацию серверных и клиентских приложений.

## Выполненные задачи

В рамках лабораторной работы было реализовано 5 задач, демонстрирующих различные аспекты сетевого программирования:

1. **UDP сервер и клиент** - Базовое взаимодействие через протокол UDP
2. **TCP сервер для решения квадратных уравнений** - Обработка данных через TCP с использованием функции `recv_until`
3. **HTTP сервер для статических файлов** - Простой веб-сервер для обслуживания HTML-файлов
4. **Многопоточный чат-сервер** - Реализация чата с поддержкой множественных клиентов
5. **HTTP сервер с формами** - Полнофункциональный веб-сервер с обработкой POST-запросов

## Используемые технологии

Python 3.x, Socket API, Threading, HTTP/1.1, JSON

## Результаты

Все задачи успешно реализованы и протестированы. В процессе работы изучены особенности различных сетевых протоколов, методы синхронизации в многопоточных приложениях и основы веб-разработки.
