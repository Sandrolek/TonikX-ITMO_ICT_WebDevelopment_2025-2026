# Задача 5 - HTTP сервер с формами

## Цель

Реализация полнофункционального HTTP сервера с поддержкой HTML форм, обработкой POST запросов и хранением данных для демонстрации создания веб-приложения с серверной логикой.

## Выполненная работа

Был создан сервер с объектно-ориентированной архитектурой, включающей классы `Request`, `Response` и `MyHTTPServer`. Сервер поддерживает GET запросы для отображения главной страницы с формой и POST запросы для обработки данных формы.

Ключевая особенность - реализация Post/Redirect/Get паттерна:

```python
# Post/Redirect/Get pattern
if req.method == "POST" and req.path == "/submit":
    # Обработка данных формы
    GRADES.append({"subject": subject, "grade": grade})
    # Редирект на главную страницу
    return Response(303, "See Other", headers=[("Location", "/")], body=b"")
```

Реализованы защита от XSS атак через HTML экранирование и валидация входящих данных.

## Результат

Успешно создан полнофункциональный веб-сервер с формами. Изучены основные принципы веб-разработки: разделение GET и POST запросов, обработка форм, валидация данных, безопасность и архитектурные паттерны веб-приложений.
