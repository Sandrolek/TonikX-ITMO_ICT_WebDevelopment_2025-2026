# Задача 2 - TCP сервер для решения квадратных уравнений

## Цель

Реализация TCP сервера для решения квадратных уравнений с демонстрацией работы с протоколом TCP и обработкой структурированных данных.

## Выполненная работа

Был создан TCP сервер, который принимает соединения от клиентов, получает коэффициенты уравнения в формате "a b c\n", вычисляет корни и отправляет результат обратно. 

Ключевая особенность реализации - функция `recv_until`, которая решает проблему потоковой природы TCP:

```python
def recv_until(sock: socket.socket, delim: bytes = b"\n", bufsize: int = 1024) -> bytes:
    buf = bytearray()
    while True:
        chunk = sock.recv(bufsize)
        if not chunk:
            break
        buf += chunk
        if delim in chunk:
            break
    return bytes(buf)
```

Алгоритм решения обрабатывает все математические случаи: линейные уравнения (a=0), два действительных корня (D>0), один корень (D=0) и комплексные корни (D<0).

## Результат

Успешно реализован TCP сервер с надежной доставкой данных. Изучены особенности TCP протокола: гарантированная доставка, контроль потока данных, но с большими накладными расходами по сравнению с UDP.
